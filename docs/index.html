Architecture

Clean modular design. Each component single responsibility. Full Synapse Client SDK stack.

sap.ts
Synapse Agent Protocol integration. Agent PDA derivation, facilitator discovery, x402 payment flow. Uses built-in SAP module.
ace-bridge.ts
Ace Data Cloud via Synapse MCP client bridge. 3 service wrappers: Search, Chat, Images. Graceful fallback to demo mode.
workflow.ts
Autonomous workflow engine. Orchestrates Search → Analyze → Visualize → Report. Full audit trail per run.
agent.ts
Main orchestrator + CLI. Manages full lifecycle: SAP init → facilitator discovery → MCP connect → execute → compliance report.
